# Security Policy

## Supported Versions

This project currently supports security fixes for the latest `main` branch.

## Reporting a Vulnerability

Please **do not** report security vulnerabilities via public GitHub issues.

Instead, report them responsibly via one of the following channels:

- Open a GitHub Security Advisory (preferred)

When reporting, please include:

- A clear description of the issue and potential impact
- Steps to reproduce (PoC if available)
- Affected versions / commit hash
- Any relevant logs or screenshots (remove sensitive data)

We will acknowledge receipt within **72 hours** and provide an estimated
timeline for remediation once confirmed.
